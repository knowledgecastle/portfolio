The Problem

The client had a PDF of client testimonials in a Canva template and wanted to transform it into a simple, mobile-friendly Notion page.

The Solution

I designed a clean Notion layout for the testimonials in five different versions, each with a slightly different structure. I tested them on mobile, recorded a video showing the client how each version displayed, and let them choose the final design.

The Process

  1. Kickoff + scope confirmation:
    Confirmed the goal, success criteria, timeline, and what “done” looks like. Listed the exact deliverables: One page for all 10 testimonials including pictures and text.
  2. Audit the current Canva file:
    Reviewed the current testimonials file on Canva. Identified the best way to transform it into Notion while being mobile-friendly.
  3. Build the core page structure:
    Created the page and moved all content from Canva to Notion. Generated 5 different versions to show the client how they would look like with small tweaks.
  4. Testing + QA:
    Checked mobile usability.
  5. Client enablement + handoff:
    Delivered a walkthrough video. Shared links of all versions with the client to check.
  6. Iteration + refinements:
    Collected feedback and made final adjustments until the page was approved.
